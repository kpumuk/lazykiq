Retries list jobs that failed and will be retried by Sidekiq.

{{< lightbox src="assets/retries.png" alt="Retries screen" >}}

## Key bindings

| Key | Description |
| --- | --- |
| `4` | Go to Retries. |
| `Up` / `k` | Move up one row. |
| `Down` / `j` | Move down one row. |
| `/` | Filter jobs by case-sensitive string. |
| `q` | Quit. |
