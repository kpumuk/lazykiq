Scheduled jobs are queued to run at a specific time.

{{< lightbox src="assets/scheduled.png" alt="Scheduled screen" >}}

## Key bindings

| Key | Description |
| --- | --- |
| `5` | Go to Scheduled. |
| `Up` / `k` | Move up one row. |
| `Down` / `j` | Move down one row. |
| `/` | Filter jobs by case-sensitive string. |
| `q` | Quit. |
